# Online Blood Bank Management System

A web-based application built using Flask and SQLite for managing blood donors, patients, and blood stock.

## Features
- Admin login
- Donor registration
- Patient blood request
- Blood stock management
- Approval system for requests

## Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite
- Gunicorn
- Railway (Deployment)

## Live Demo
https://blood-bank-management-production.up.railway.app
